# Google PT Opener

A simple Chrome extension that opens www.google.pt in a new tab when you click on the extension popup.

## Features

- Opens www.google.pt when the popup is clicked
- Simple, user-friendly interface
- Lightweight and fast

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right corner
3. Click "Load unpacked"
4. Navigate to and select the `tutorial/google-pt-opener` directory
5. The extension will now appear in your extensions toolbar

## Usage

1. Click on the "Google PT Opener" extension icon in your Chrome toolbar
2. A popup will appear with instructions
3. Click anywhere in the popup to open www.google.pt in a new tab

## Files

- `manifest.json` - Extension configuration
- `popup.html` - Popup UI
- `popup.js` - Extension logic

## Permissions

- `tabs` - Required to create new tabs
